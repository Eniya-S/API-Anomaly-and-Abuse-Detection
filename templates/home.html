Machine Learning Based API Anomaly Detection

This Flask application generates API access logs that will be used later for Machine Learning based anomaly detection.

Available API Endpoints

Endpoint Method Description
/signup POST Create New User
/login POST User Login
/products GET View API Services
/profile GET View User Profile
/dashboard GET View Dashboard Statistics
/change-password POST Update User Password
/api-test POST Generate API Test Requests
/logout GET Logout User
